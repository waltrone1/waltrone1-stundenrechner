INNO-SETUP-ANLEITUNG FUER waltrone1-Stundenrechner
==================================================

VORAUSSETZUNG IM PROJEKTORDNER
------------------------------
Beispiel:
C:\Users\Alexander Cobucci\OneDrive\Desktop\TESTUMGEBUNG\Stundenrechner\V1

Mindestens vorhanden:
- .\waltrone1-stundenrechner.py
- .\waltrone1-Stundenrechner.ico


1) VENV ANLEGEN
---------------
py -3 -m venv .venv
.\.venv\Scripts\python.exe -m pip install -U pip


2) PAKETE INSTALLIEREN
----------------------
.\.venv\Scripts\python.exe -m pip install -U pip setuptools wheel pyinstaller customtkinter pillow tkcalendar babel


3) EXE MIT PYINSTALLER BAUEN
----------------------------
.\.venv\Scripts\python.exe -m PyInstaller `
  --noconfirm `
  --clean `
  --name "waltrone1-Stundenrechner" `
  --icon ".\waltrone1-Stundenrechner.ico" `
  --paths "." `
  --hidden-import "customtkinter" `
  --hidden-import "tkcalendar" `
  --hidden-import "babel.numbers" `
  --hidden-import "babel.dates" `
  --hidden-import "PIL" `
  --hidden-import "PIL.Image" `
  --hidden-import "PIL.ImageTk" `
  --hidden-import "PIL.IcoImagePlugin" `
  --collect-submodules "customtkinter" `
  --collect-submodules "tkcalendar" `
  --collect-submodules "PIL" `
  --windowed `
  ".\waltrone1-stundenrechner.py"

ERGEBNIS
--------
.\dist\waltrone1-Stundenrechner\

Wichtig:
Diesen kompletten Ordner weiterverwenden, nicht nur die EXE.


4) VORAUSSETZUNG IM BUILD-ORDNER
--------------------------------
Beispiel:
C:\Build

Struktur:
C:\Build
├─ waltrone1-Stundenrechner
│  ├─ _internal
│  └─ waltrone1-Stundenrechner.exe
├─ waltrone1-Stundenrechner.iss
└─ waltrone1-Stundenrechner.ico


5) INNO-SETUP-SCRIPT
--------------------
Datei:
C:\Build\waltrone1-Stundenrechner.iss
